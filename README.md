# TicTacToeSwiftUI
This project can be used for implementation of Tic Tac Toe Game with help of SwiftUI


## Three Case Game
```
First Case : 'Player X is Winner'
Second Case : 'Player O is Winner'
Third Case : 'GameOver : Match Tied'
```

## Color Scheme 
From Dribble : `Tic Tac Toe`

## Color in XCAssets
```
1. colorBackground
2. color 1
3. Some Extra Assets
```
